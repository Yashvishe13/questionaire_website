# General Instructions

In this study, you will evaluate edited versions of music clips.

Each question includes:

- one **original music clip**,
- one **editing instruction**,
- two **edited versions** (Version A and Version B).

## Your Task

For each question, choose **which edited version better preserves a specific musical aspect of the original**, while ignoring the intended edit.

## Important Guidelines

- Focus only on the specified musical aspect.
- Ignore whether the edit itself is correct.
- Do NOT judge based on:
  - personal preference
  - audio quality
  - loudness
  - or which version sounds more interesting

Instead, select the version that introduces **fewer unintended changes**.

## Response Options

- **A is better**
- **B is better**
- **About the same**

You may listen to each audio multiple times.
